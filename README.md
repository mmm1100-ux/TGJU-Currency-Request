# TGJU Currency Request — پلاگین وردپرس

یک پلاگین سبک وردپرس برای «محاسبه‌ی لحظه‌ای تبدیل ارز به تومان» و «ثبت درخواست کاربر» به‌عنوان لید (Lead)؛ مناسب سایت‌های خدمات ارزی/مهاجرت/مشاوره که نیاز به جمع‌آوری درخواست و پیگیری دارند. 

---

## خلاصه قابلیت‌ها
- ارائه فرم تبدیل ارز از طریق شورت‌کد (قابل استفاده در برگه/نوشته و صفحه‌سازها). 
- محاسبه‌ی نتیجه تبدیل به‌صورت زنده با AJAX (بدون رفرش صفحه). 
- جریان ارسال درخواست:
  - نمایش مودال راهنما برای ورود در صورت لاگین نبودن کاربر.
  - دریافت اطلاعات تماس و ثبت درخواست برای پیگیری. 
- پنل مدیریت اختصاصی:
  - تنظیم ایمیل دریافت اعلان‌ها
  - مشاهده لیست درخواست‌های ثبت‌شده در داشبورد. 

---

## نکات فنی
- توسعه مبتنی بر استانداردهای وردپرس:
  - پیاده‌سازی Shortcode برای Embed شدن آسان. 
  - استفاده از WordPress AJAX برای عملیات تبدیل و ثبت درخواست. 
- امنیت و کیفیت ورودی‌ها:
  - استفاده از nonce برای جلوگیری از درخواست‌های جعلی (CSRF).
  - پاک‌سازی/اعتبارسنجی داده‌های ورودی پیش از پردازش و ذخیره‌سازی. 
- طراحی تجربه کاربری:
  - رابط کاربری ریسپانسیو و سازگار با موبایل.
  - مودال‌های مجزا برای ورود، تکمیل اطلاعات و پیام موفقیت/خطا. 
- ذخیره‌سازی و پیگیری:
  - ثبت درخواست‌ها به‌صورت ساختاریافته برای مشاهده و پیگیری توسط مدیر سایت. 

---

## نحوه استفاده (برای دمو)
پس از نصب و فعال‌سازی، شورت‌کد زیر را در صفحه قرار دهید: 
[tgjucurrencyrequest]

نمونه با برچسب سفارشی برای فیلد کشور: 
[tgjucurrencyrequest countrylabel="کشور"]




## موارد استفاده
جمع‌آوری لید برای خدمات ارزی، حواله، مهاجرت و مشاوره. 
ارائه «تخمین سریع» تبدیل ارز برای کاربران قبل از تماس/ثبت درخواست.
